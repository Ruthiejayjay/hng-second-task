### Local development

Api url: http://localhost:8000/api

---
<strong>Clone Project:</strong>
```
git clone https://github.com/Ruthiejayjay/hng-second-task
```
---
<strong>Starting server:</strong>

Open new terminal in project and run the following
```
cd hng-second-task

cp .env.example .env

composer install

php artisan migrate

php artisan serve
```
